# DI-1240 — SapHanaWindowFunctionsMemberTranslator comment contradicts its own PERCENTILE_CONT/DISC flags

**Severity**: low  **Category**: doc-gap  **Status**: open

## Location
- `Source/LinqToDB/Internal/DataProvider/SapHana/Translation/SapHanaMemberTranslator.cs:414-417` — capability-flag block for `SapHanaWindowFunctionsMemberTranslator`

## Pattern matched
No literal regex in the kb-issue-categories.md catalog covers "inline comment contradicts an adjacent capability-flag value"; this is flagged under `doc-gap` by category intent (a mis-documented invariant), not a catalog regex hit — see AUDIT-NOTE below proposing a catalog entry. The comment on lines 416-417 states "windowed PERCENTILE_CONT/DISC ... execute on SAP HANA", but the two flags it sits directly beside — `IsPercentileContSupported` (line 414) and `IsPercentileDiscSupported` (line 415) — are both `false`. The area's own architecture INDEX (`.agents/knowledge-base/areas/PROV-SAPHANA/INDEX.md`, "Known issues / debt") independently flagged this exact contradiction at the same `currentSha` without resolving it, confirming this isn't a misreading.

## Why it matters
A maintainer who trusts the comment could flip the two flags to `true` expecting a working windowed-percentile translation on SAP HANA and ship a runtime SQL failure, or could waste time re-investigating something the comment claims already works. Capability flags in this file gate real SQL generation for every consumer of `SapHanaMemberTranslator`'s window-function path, so a wrong inline claim next to the flags it describes is a direct trap for the next edit in this area.

## Suggested fix
Verify against a live SAP HANA instance (or authoritative SAP HANA SQL reference) whether windowed `PERCENTILE_CONT`/`PERCENTILE_DISC` actually execute. If they do, flip `IsPercentileContSupported`/`IsPercentileDiscSupported` to `true` and add a regression test covering the windowed form. If they don't, edit the comment to drop the PERCENTILE_CONT/DISC claim so it stops contradicting the flags immediately below it.
