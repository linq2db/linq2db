# 2011

## Releases this year

- No version-tagged releases this year

## Top themes

- **Project genesis and initial code import** ‚Äî The repository was created on 2011-06-18 with a single README commit (`036e6de`). A dormant period of six months followed. On 2011-12-13 the project came alive with a 541-file commit (`6f5bb03`) that imported the entire initial codebase: SQL create scripts for Access, DB2, Firebird, Informix, MySQL, Oracle, PostgreSQL, SQLite, SqlCe, and Sybase; data provider classes under Source/Data/DataProvider/; the LINQ query builder under Source/Data/Linq/Builder/; mapping and reflection infrastructure; and redistributable driver DLLs. Two earlier commits in June (`058f2a7`, `fe43170`) had seeded initial DataTypes work absorbed into this import.

- **Large-scale refactoring to strip legacy BLToolkit code** ‚Äî Starting on 2011-12-14 and running through 2011-12-16, a dense wave of refactoring commits collectively deleted over 25 000 lines. Commits `391c086`, `61a166c`, `e5f5e39`, `fd42b67`, `c3c8c9c`, `ad4e1fa`, and `6a62586` removed legacy BLToolkit-era classes (BLToolkitSection.cs renamed to LinqToDBSection.cs in `e5f5e39`; DbManager stripped in `391c086`) and trimmed DataAccess and Mapping code no longer needed in the LINQ-centric model. Two mass-rename commits (`a819c6f` at 127 files, `21ee2d9` at 150 files) reorganized provider and builder files, flattening Source/Data/DataProvider/ and moving CompiledQuery.cs to Source/.

- **Nemerle port alongside the C# codebase** ‚Äî The project maintained a parallel Nemerle (.n) implementation of the SQL builder and data-provider layer. VladD2 added the Nemerle project file and build script in `efef00b`, and AssemblyInfo.n in `5a51837`. Two commits on 2011-12-20 (`5cc5b1a` by VladD2, `870c8ab` by VladD2) fixed compatibility with Nemerle 1.1. Macro work continued through late December: TypeCode macro introduced in `3de5f84` and refactored in `04e0303` and `11f8876`; ReplaceCtorParameter macro refactored in `55f791f`.

- **SqlExpr development sprint (late December)** ‚Äî From 2011-12-22 to 2011-12-27, eight commits share the subject "Working on SqlExpr." (`44f6520`, `059859e`, `8fa4d1d`, `dd3de4f`, `6fd64f0`, `1e663e4`, `ef7a767`, `2f4025c`). These built up Source/SqlBuilder/SqlExpr.n and related files; commit `44f6520` alone touched 14 files with +610 lines. Branch 122 was merged three times (`2a1cdce`, `9b1930b`, `efe14de`), indicating the SqlExpr work ran on a feature branch before being integrated.
