## Top themes

- **General code modernization** -- 4c2d02e (FC:1155, Stuart Turner, Mar 9) applied general code modernization across the codebase (#5201), a 53 409-insertion / 59 088-deletion pass updating language idioms, removing legacy patterns, and adopting C# modern features. The largest commit of the partial year by file count.

- **Packaging migration** -- 70157bd (FC:110, Stuart Turner, Jan 30) migrated NuGet packaging from .nuspec files to .csproj-native packaging (#5279), consolidating project metadata and simplifying the publish pipeline.

- **SQL generation and query optimizer fixes** -- Svyatoslav Danyliv drove a concentrated fix stream: 020ef16 (FC:29, #5307), bb2ded1 (FC:30, #5285), b178e97 (FC:10, #5321), 557df92 (FC:21, #5344), 0778057 (FC:2, #5262), 6655c3c (FC:27, #5270), 157f8d6 (FC:48, #5284). Follow-on: 329fc04 (May 9, #5510), 4a478ff (May 11, #5511), 2e67baf (May 29, fix #5125 do not mutate shared AST nodes in column-nesting corrector, #5556), 5002e58 (May 26, fix #5547 resolve association when Select projection drifts DeclaringType, #5548), edd8926 (May, fix #5505 skip ValueConverter wrap when SET value uses a field, #5506). 7ed24e5 (Jun 7, #5577) fixed MapMember/ConvertMember mappings surviving unexpanded through the initial expose pass. fce0ec5 (Jun 9, #5558) fixed InvalidCastException when MovingOuterPredicateVisitor treated a nested SelectQuery as a hoistable scalar. b3340aa (Jun 13, #5587) fixed spurious [item] column when a LEFT JOIN re-expanded an in-memory AsQueryable() source with a decimal projection descriptor still ambient.

- **CTE and UPDATE/DELETE improvements** -- 1d66ab9 (Mar 1) added CTE support for UPDATE and DELETE statements (#5385). c2246d0 (FC:32, Apr 20) added the MATERIALIZED CTE hint via the AsCte builder (#5323/#5478). 3a156a8 (FC:9, Apr 20) improved UPDATE transformation for providers that do not support UPDATE FROM (#5414). b7b9a5f (May, #5507) fixed correlated row-setter UPDATE on Oracle 12+.

- **ClickHouse.Driver migration** -- 529889b (FC:10, Mar 6) added support for ClickHouse.Driver 1.0 provider (#5401). f6799ed (FC:8, Jan 6) updated the driver package (#5227). Follow-up fixes: 8cc54ca (May 7, #5449), 7d5e62e (May 22, #5553). c21e629 (Jun 12, #5608) updated Octonica.ClickHouseClient to 4.1.4 and re-enabled the Octonica CI matrix entry disabled in #5460.

- **.NET new LINQ methods and cast improvements** -- 956f5fa (FC:21, Mar 7) added support for new .NET LINQ methods (#5369). 6b2186c (FC:2, Mar 5) added Cast operator support (#5337). 0a7e37f implemented .NET 10 LeftJoin and RightJoin operators (#5233).

- **String translation expansion** -- de4d000 (FC:89, May, #5504) wired string.Concat translation across all providers. c35da70 (FC:17, May, #5515) added string.TrimStart / string.TrimEnd translation. 5999b4f (Jun 9, #5544) migrated string.IsNullOrWhiteSpace from 11 IExtensionCallBuilder classes to a virtual TranslateIsNullOrWhiteSpace on StringMemberTranslatorBase.

- **NULLS FIRST/LAST ordering support** -- 2f99880 (Jun 5, #5561) added Sql.NullsPosition and new OrderBy/ThenBy overloads, rendering natively on 8 providers; emulated via CASE WHEN on others. Also added DataOptions.SqlOptions.DefaultNullsPosition for a connection-level default and DefaultNullsOrdering for natural null sort order tracking.

- **Aggregate and subquery fixes** -- 75fce59 (FC:24, May, #5502) wrapped non-nullable subquery aggregates with COALESCE. 7733aead (May 29, #5557) fixed Count/Sum/Min/Max/Average translation for closure-captured collection expressions. e967c53 (May, #5522) fixed IN-to-EXISTS conversion for set-based subqueries. 4b252f7 (Jun 4, #5568) fixed nullable-result SUM null-guard gating on argument nullability instead of result type, causing Nvl(x, NULL) on Informix.

- **DuckDB provider** -- 3537e02 (May 10, #5451) added DuckDB as a new supported data provider. f6d511c (May 11, #5518) corrected a date translation regression.

- **Timestamp and timezone translation** -- d779808 (May 10, #5467) reworked now-family translation, introducing Sql.CurrentTimestampUtc and Sql.CurrentTzTimestamp. c3f260f (May 11, #5309/#5517) fixed DateTime.Date truncation. 394e1df (Jun 5, #5466) corrected DateTimeOffset-to-zoneless-column literal translation, normalizing to UTC and emitting a CAST.

- **AsQueryable parameterization API** -- bda0798 (May 10, #5495) added a parameterization API for AsQueryable, exposing IAsQueryableBuilder and a three-argument LinqExtensions.AsQueryable overload.

- **Enum.HasFlag translation** -- 9272e49 (May 10, #5503) added LINQ-to-SQL translation for Enum.HasFlag via bitwise AND comparisons.

- **BulkCopy and provider improvements** -- 1c80c3e (FC:9, Mar 20) added IgnoreConflicts for MySql/MariaDB, PostgreSQL, and SQLite BulkCopy (#5395). e589e33 (FC:6, Feb 26) added PostgreSQL bulk copy timeout support (#5377). 5a0e28b (FC:17, Apr 24) corrected DropTable throwExceptionIfNotExists behavior (#5463).

- **Correlated subquery validation** -- b12201c (Jun 4, #5574) extended SqlQueryValidatorVisitor to reject correlated subqueries in predicate/expression position for level-0 providers (ClickHouse, YDB), gated by a ForErrorReporting flag to preserve optimizer/eager-load probes. 48771ed (Jun 4, #5582) fixed NOT IN/IN emulation null-safety for level-0 providers.

- **PublishSingleFile support** -- a94229f (Jun 11, #5489) replaced Assembly.Location-dependent helpers with Tools.IsProviderAssemblyPresent and added a Tests.SingleFile CI smoke test. 034c150 (Jun 12, #5607) moved the smoke test out of the test pipeline.

- **EF Core many-to-many associations** -- bb5b975 (Jun 10, #5588) added translation for EF Core skip-navigation associations in EFCoreMetadataReader, handling composite keys, self-references, field-mapped keys, and shadow principal keys.

- **linq2db.cli per-RID packaging** -- 4a6efbf (FC:11, May 18, #5539) replaced the single omnibus CLI package (hit nuget.org 250 MB ceiling at ~416 MB during 6.3.0) with per-RID packages using .NET 10 SDK ToolPackageRuntimeIdentifiers. Seven RIDs shipped.

- **Native AOT and cross-platform fixes** -- 6339cd4 (May 24, #5552) fixed MemberInfoEqualityComparer crash on Native AOT. 060d4d9 (May 29, #5546) fixed SqlTransparentExpression static constructor fatal on Mono/Android AOT.

- **Release prep 6.3.0** -- 0d283ac (FC:111, May 16, #5533) covered 37 package updates, PublicAPI reconciliation, SqlClient 7.0.1 fixes, and T4 scaffold packaging corrections.

- **Async code analyzers and review tooling** -- 68583d3 (FC:11, Mar 6) added async code analyzers (#5402). 0e6f146 (FC:39, Apr 22) review tooling improvements (#5479). 0ca48b4 (Jun 4, #5573) resolved deferred Meziantou.Analyzer rules from #5532: enabled MA0150 and MA0111, fixed MA0186 (82 Equals parameters annotated [NotNullWhen(true)]), permanent justifications for 7 rules.

- **Build configuration gating** -- 81db2d7 (May 9, #5516) restricted IDE analyzers, ReportAnalyzer, and XML-doc generation to Release builds only.

- **CI and infra improvements** -- 161fcac (Jun 2, #5563) stabilized Linux/macOS DB2/Informix native client loading via SetDllImportResolver. a12128c (Jun 2, #5572) reset baselines working tree before retried Oracle/Access test attempts. 034c150 (Jun 12, #5607) scoped PublishSingleFile smoke test to build pipeline only. 1020a48 (Jun 10, #5603) switched remote test servers to dynamic per-process ports. 171e3c3 (Jun 13, #5610) added live test-progress heartbeat (LINQ2DB_TEST_PROGRESS env var, JSON file, /test-progress skill).

- **String concat follow-on fixes** -- d70be94 (Jun 4, #5566) fixed bogus IS NULL guards on non-null Sybase concat operands. db07543 (Jun 7, #5569) migrated Sybase ASE to ANSI || (from T-SQL +). 4ba40b5 (Jun 2, #5567) restored IS NULL pushdown through SqlConcatExpression. 1ae9418 (Jun 8, #5570) collapsed redundant case-conversion wraps in Guid-to-string translation.

- **MERGE and mapping fixes** -- 66b9d06 (Jun 12, #5545) fixed nested fluent column mapping in MERGE implicit setter via GetMemberAccessExpression, handling explicit interface implementations and inheritance discriminators.

- **LINQPad driver and scaffold fixes** -- d159f54 (Jun 4, #5571) fixed the LINQPad driver package shipping under lib/net8.0-windows7.0/ since #5279 (broken on macOS/Linux since 6.2.0). 317dfaa (Jun 2, #5565) fixed scaffold packages failing to resolve Redist DLL references outside the full solution build. 95e594f (Jun 3, #5579) fixed LINQPad ComboBoxes overflowing in the connection dialog.

- **DateTime subtraction in projection** -- 8d2da5f (Jun 12, #5581) changed bare DateTime/DateTimeOffset subtraction (TimeSpan result) to evaluate client-side; (a-b).TotalX still maps to Sql.DateDiff.

- **Claude Code KB-infrastructure (infra/claude branch)** -- A parallel stream on the infra/claude branch added Claude Code agent tooling and a persistent knowledge-base build pipeline. Key commits: 8b1905f (Apr 8, initial Claude Code setup, #5452), 30e4333 (FC:25, Apr 20, Claude Code configuration and tuning, #5473), a198cb8 (FC:40, Apr 26, KB infrastructure + step 3 progress), f8e444c (FC:50, May 4, kb work), a2cdbc4 (May 3, pin per-agent model), d1576fe (May 5, .claude/ refactor + /chores dispatcher), 810c172 (May 6, edit-gh-comment.ps1 + agent-rules guardrails), d52a4ed (May 7, kb-build drain), 3b1f19f (FC:1030, May, KB build infrastructure + azp-run helper, #5487).
