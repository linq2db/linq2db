# 2024

## Releases this year

- Version 6.0 -- May 2024 -- 0cb7676 merged the version_6 branch into master on May 29. 782a40b (release preparation, FC:29) preceded it. The version_6 branch held the major LINQ query parsing rewrite (7ec71d5) gated for the v6 boundary.
- Post-6.0 regression fixes -- Jun-Dec 2024 -- 193926e (Version 6 regressions Part 3, #4530), b5c4485 (#4527), 80c54cd (Release chores, #4525). Regression stream continued through the final commit (64443a2, Dec 27).
- .NET 9 / EF Core 9 -- Nov 2024 -- b74e5de (FC:119, #4693).

## Top themes

- **LINQ query parsing refactor (v6 gate)** -- 7ec71d5 (FC:890, Svyatoslav Danyliv, May 29) refactored LINQ query parsing (#3401), a 71 968-insertion / 45 285-deletion rewrite that formed the structural core of version 6. Work on the version_6 branch spanned multiple months of merge commits before the final integration.

- **SQL generation refactor** -- b7de32b (FC:688, Svyatoslav Danyliv, Dec 20) refactored SQL generation with additional bug fixes (#4676), a 34 493-insertion / 17 913-deletion commit arriving at year-end and generating a further round of regression fixes into early 2025.

- **Provider configuration and versioning** -- 4704ace (FC:232, MaceWindu, Apr) refactored provider configuration and versioning (#4002), reorganizing how provider versions are declared and consumed across all supported databases.

- **EF Core repo consolidation** -- 0c99d98 (FC:321, Sep) moved linq2db.EntityFrameworkCore into the main repository (#4595), adding approximately 50 993 lines of EF Core integration code and tests.

- **TFM updates** -- f973e5b (FC:502, Jan) updated solution TFMs (#4370), removing older targets and adding net8.0. b74e5de (FC:119, Nov) added .NET 9 and EF Core 9 (#4693).

- **NUnit 4 migration** -- 48c35f2 (FC:318, Apr) migrated the entire test suite from NUnit 3 to NUnit 4 (#4451).

- **API baselines** -- 56edce9 (FC:52, Jan) added initial API baseline files (#4369), establishing the ApiCompat check tooling. 9b9b4d6 (Feb) moved to version-less baseline folders (#4411).

- **Metrics and observability** -- 71b9bd8 (FC:90, Igor Tkachev, Jan) added .NET metrics integration (#4005). f45b862 (FC:20, Igor Tkachev, Jul) added activity tags (#4563).

- **Post-v6 regression cadence** -- From June through December the primary work stream was stabilization: 193926e, b5c4485, c19b6cf, f97a3ab, 2300da4, 46cdda9, 0246b63, 386b16c, fcbe0c5, and 64443a2. The high regression count is consistent with the scope of 7ec71d5.
