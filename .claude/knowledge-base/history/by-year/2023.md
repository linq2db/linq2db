# 2023

## Releases this year

- 5.2.0 -- May 2023 -- 9c6431c fixed the 5.2.0 merge; a1a88d4 set release version. Patch bump-version commits in Feb, Mar, Apr, May.
- 5.3.0 / 5.x -- Oct 2023 -- c78d04d bumped version; 6269048 merged the release branch into master.
- Full year ran on the 5.x line; no 6.x branch signals in commit subjects.

## Top themes

- **Fluent mappings generator** -- b2ad630 (FC:1427, Mar 15, MaceWindu) added the fluent mappings generator (#4021), a 98 759-insertion commit generating complete mapping code from schema reflection. Largest commit of the year by insertion count.

- **Options refactoring** -- b6d5f78 (FC:411, Svyatoslav Danyliv, Jan 11) refactored options handling (#3530), an 11 667-insertion / 6 165-deletion rewrite distributing configuration across typed option classes. 738fd90 (FC:72, Feb) moved EntityDescriptorCreatedCallback to options (#3969). 380d526 (FC:38, Jan) added new connection setup options (#3939). f90d762 (FC:66, Jan, resubmit of #3933) introduced explicit breaking changes to the fluent configuration API.

- **Attributes refactor** -- 13fbe5f (FC:218, Jan 23) refactored how mapping attributes are read and applied (#3900), touching 3 822 insertions / 3 459 deletions across the attribute resolution path.

- **SqlBuilder pooling and performance** -- c750374 (FC:49, Jan 23) added SqlBuilder pooling (#3919) to reduce per-query allocations. 27f5156 (FC:14, Oct) optimized EntityDescriptor internals (#4276). 3cff244 (FC:42, Sep) fixed context leaking into the query cache (#4229).

- **Temporal table support** -- 7178ca2 (FC:25, Igor Tkachev, Mar) added SQL Server temporal table support (#4001) for time-travel queries.

- **MySQL, ClickHouse, and hint fixes** -- ce3f03d (FC:21, Igor Tkachev, Jan) added MySQL lock hints (#3932). 08dd7eb (FC:24, Igor Tkachev, Feb) switched ClickHouse from EXISTS to IN since ClickHouse does not support EXISTS (#3966). 5b724c7 (FC:18, Igor Tkachev, Aug) fixed hints for UNION (#4202). ClickHouse query hints/extensions: 99511fe (FC:17, May, #4075).

- **Dependency and infrastructure maintenance** -- 3a1a473 (FC:273, Aug 30) updated all dependencies and fixed analyzer warnings (#4251). 368aee2 (FC:78, Dec) migrated CI testing to custom Docker images (#4345).

- **NRT and nullability improvements** -- 14ee4c8 (FC:15, jods, Jan) added C# nullable reference type support (#3606). cd6461f (FC:2, Shane Krueger, May) fixed outer apply column nullability tracking (#4092). 7011314 (FC:2, Igor Tkachev, Jan) fixed NRT annotations on eager load APIs (#3924).

- **Parameter handling** -- 8f03714 (FC:36, Jun) implemented parameter name normalization (#4146). 71da98f (FC:39, May) fixed procedure out-parameter extraction (#4128) after a revert cycle. 9db7167 (FC:4, Shane Krueger, Sep) added tests and fixed UniqueParametersNormalizer (#4230).
