# Linq to DB extensions for NHibernate

`linq2db.NHibernate` lets you execute [Linq to DB](https://github.com/linq2db/linq2db) (linq2db) queries
using an existing NHibernate `ISession`, reusing NHibernate's mapping metadata and its open connection.

> Work in progress — being modernized against linq2db 6.x and NHibernate 5.x.
